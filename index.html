Módulo de Fundamentos

1. Introducción a Python para Data Science (2 ECTS)

Explora las bases de Python, las librerías fundamentales como NumPy, Pandas y Matplotlib, y desarrolla habilidades esenciales para análisis de datos.

2. Fundamentos de Machine Learning (2 ECTS)

Aprende los conceptos básicos del aprendizaje supervisado y no supervisado, y utiliza herramientas como Scikit-learn para construir modelos.

3. Fundamentos de Deep Learning (2 ECTS)

Descubre las redes neuronales y frameworks como TensorFlow y PyTorch para crear tus propios modelos de aprendizaje profundo.

Módulo de Aplicaciones

4. Aprendizaje por Refuerzo Profundo

Desarrolla agentes inteligentes que aprenden a interactuar con su entorno para resolver problemas complejos en mecánica y aeroespacial.

5. Machine Learning Científico y AI para la Ciencia

Aplica herramientas avanzadas de ML para simulaciones científicas y optimización de procesos en ingeniería.

6. AI Generativa

Explora cómo generar nuevos diseños, datos sintéticos y simulaciones a través de técnicas avanzadas de AI generativa.
